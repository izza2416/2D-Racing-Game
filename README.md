# 🏎️ 2D Racing Game

A fast-paced 2D racing game built in **C++** using the **SFML** graphics library. 
Race against an AI opponent, dodge obstacles, and cross the finish line first!

---

## 🎮 Gameplay
- Control your **blue car** using keyboard keys
- Race against a **green AI car**
- Avoid **orange cone obstacles** on the track
- First car to reach the **finish line** wins!

---

## 🕹️ Controls
| Key | Action |
|-----|--------|
| `W` | Move Up |
| `S` | Move Down |
| `A` | Move Left |
| `D` | Move Right |

---

## ✨ Features
- 🚗 Player vs AI racing
- 🏁 Animated scrolling road
- 🚧 Obstacle detection and collision handling
- 📁 Game result saved to `gamedata.txt`
- 🎨 Graphical cars, cones, boundaries and finish line



## 🛠️ Built With
C++
SFML 2.x ( Graphics)
